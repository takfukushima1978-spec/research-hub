CLAUDE CODE 公式発表

100万トークンが解禁！
Opus 4.6の真価が発揮される

2026年3月13日 ｜ v2.1.75 1Mコンテキスト正式対応
Claude Code v2.1.75で、Opus 4.6の100万トークンコンテキストウィンドウがMax/Team/Enterpriseプランで正式利用可能に。同時に/colorコマンドでセッションの色分けカスタマイズも追加。メモリファイルへのタイムスタンプ機能など、実用的な改善が目白押しだ。
1

1Mコンテキストの正式解禁

2月のOpus 4.6リリース時はベータだった100万トークンのコンテキストが、Max/Team/Enterpriseプランで正式に利用可能に。大規模なモノレポ全体をコンテキストに載せて、横断的な修正や分析ができるようになった。

💡 数十万行規模のプロジェクトでも「コンテキスト不足」を気にせず作業できる。アーキテクチャの理解、依存関係の把握、大規模リファクタリングで真価を発揮する。
📎 Claude Code Changelog
2

/color ── セッションの見た目をカスタマイズ

/colorコマンドで、Claude Codeのセッションごとに色をカスタマイズできるようになった。複数のターミナルでClaude Codeを同時に使う場面（Agent Teamsなど）で、どのセッションがどのタスクかを視覚的に区別できる。

📎 Claude Code Changelog

📝 編集部コメント

100万トークンの正式解禁は、Claude Codeの使い方のパラダイムシフトだ。「コンテキストを節約する」から「コンテキストに全部載せる」へ。もちろんコストとのトレードオフはあるが、バグの見落としや文脈の断絶が減ることを考えれば、投資対効果は高い。
