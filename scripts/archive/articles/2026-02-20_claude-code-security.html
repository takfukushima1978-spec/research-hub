CLAUDE CODE 公式発表

AIがコードの脆弱性を自動検出
Claude Code Security登場

2026年2月20日 ｜ Claude Code Security
Anthropicが「Claude Code Security」を限定リサーチプレビューとして公開。コードベースのセキュリティ脆弱性をAIがスキャンし、具体的な修正パッチまで提案する。従来のSAST/DASTツールでは見つけられなかった問題を発見できるという。
1

従来ツールの限界を超える

Claude Code Securityは、単なるパターンマッチではなく、コードのロジックを理解した上で脆弱性を検出する。ビジネスロジックの欠陥、認証フローの問題、複数ファイルにまたがるセキュリティホールなど、従来のツールが苦手とする領域をカバーする。

💡 Claude Code Web版で利用可能。既存のCI/CDパイプラインに組み込むことで、プルリクエストごとにセキュリティチェックを自動実行できる。
📎 公式発表を読む（Claude Code Security）
2

修正パッチの自動提案

脆弱性を見つけるだけでなく、具体的な修正パッチを提案してくれる点が画期的だ。人間のレビュアーは「この修正で本当に問題が解決するか」を確認するだけでよい。セキュリティの専門知識がないチームでも、高い水準のセキュリティ対策が可能になる。

📎 公式発表を読む（Claude Code Security）

📝 編集部コメント

セキュリティは「後から対処」ではコストが跳ね上がる領域。Claude Code Securityのようなツールが開発フローに組み込まれることで、「シフトレフト」（早期発見・早期修正）が現実的になる。4月のProject Glasswingへの伏線としても注目すべき発表だ。
