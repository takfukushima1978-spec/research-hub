Voice 10言語追加 & /claude-api

2026年3月5日 ｜ v2.1.69
Claude Code v2.1.69で、Voice Modeに10の新言語が追加され合計20言語に対応。さらに/claude-apiスキルでAPI開発が効率化。プラグインの再読み込みコマンドやGo開発向けのネットワーク設定改善など、グローバルな開発者に向けた改善が多数。
1

Voice Mode ── 20言語対応に

Voice Modeで新たに10言語が追加され、合計20言語に対応。日本語を含むアジア言語のサポートも強化され、音声でClaude Codeを操作する体験がグローバルに広がった。

スペースバーを押して話すプッシュトゥトーク方式で、手がふさがっている時や、素早くアイデアを伝えたい時に便利だ。

💡 /voiceコマンドで起動。スペースバー長押しで話し、離すと送信。20言語対応で非英語話者にも実用的に。

/claude-api スキル

Anthropic API開発を支援する組み込みスキル/claude-apiが追加。APIのエンドポイント仕様、ベストプラクティス、サンプルコードをClaude Codeのコンテキストに直接読み込める。

📝 編集部コメント

Voice Modeの多言語化は、Claude Codeが「英語圏の開発ツール」から「グローバルな開発プラットフォーム」へ進化する重要な一歩。日本語でのVoice操作は、日本の開発者にとって参入障壁を大幅に下げてくれる。
