CLAUDE CODE 公式発表

サードパーティ締め出し？
Claude Codeのポリシー変更の真意

2026年4月4日 ｜ サードパーティツールポリシー
Anthropicが突如発表したポリシー変更が波紋を広げている。OpenClawをはじめとするサードパーティハーネスでの「サブスクリプション枠」によるClaude Code利用が終了。従量課金への移行が必須となる。その背景にある業界の動きとは。

何が変わるのか

これまで、Claude Codeはサードパーティ製のハーネス（ラッパーツール）からもサブスクリプション枠を使ってアクセスできていた。特に人気だったのがオープンソースの「OpenClaw」だ。

今回の変更で、サードパーティハーネス経由での利用はAPI従量課金のみとなる。つまり、月額固定のサブスクリプション枠では外部ツール経由で使えなくなる。

💡 公式のClaude Code CLIを直接使うユーザーには影響なし。サードパーティツール経由で使っていた場合のみ対応が必要。

移行クレジットの提供

急な変更に対する配慮として、Anthropicは影響を受けるユーザーに移行クレジットを提供すると発表。従量課金への移行をスムーズにするための措置だ。

クレジットの申請方法や金額の詳細は、対象ユーザーに個別通知される予定。

背景 ── OpenClaw創設者のOpenAI移籍

このポリシー変更の背景には、OpenClawの創設者がOpenAIに移籍したという業界の動きがあるとされている。競合他社のツールを通じてサブスクリプション枠が使われる状況を、Anthropicが見過ごせなくなったと見られている。

⚠️ AIエージェントツール市場は今後も再編が進む可能性が高い。特定のサードパーティツールに依存しすぎないよう、公式CLIの使い方もしっかり押さえておきたい。

📝 編集部コメント

プラットフォーマーとサードパーティの関係性は、AIツール市場でも避けて通れないテーマだ。今回のポリシー変更は、Claude Codeが単なる開発ツールから「エコシステム」へと成熟しつつある証拠とも言える。ユーザーとしては、公式CLIの習熟度を上げておくのが最善の備えだろう。
