AIハッカー「Project Glasswing」の衝撃

2026年4月7日 ｜ Project Glasswing & Claude Mythos Preview
Anthropicが発表した「Project Glasswing」が、サイバーセキュリティ業界に衝撃を与えている。Claude Codeを活用した自律的な脆弱性探索エージェントが、OpenBSDに27年間潜んでいたバグやLinuxカーネルの特権昇格チェーンを発見。同時に新モデル「Claude Mythos Preview」もお披露目された。
1

Project Glasswing ── AIが自律的にバグを探す

Project Glasswingは、Claude Codeのエージェント機能をサイバーセキュリティに応用したプロジェクトだ。AIエージェントがソースコードを自律的に読み、脆弱性を発見し、概念実証（PoC）まで作成する。

人間のセキュリティ研究者が何年もかけて見つけられなかったバグを、AIが数時間で見つけてしまう。これはセキュリティの世界のパラダイムシフトと言っていい。

📎 公式発表を読む（Project Glasswing）

発見された主な脆弱性

Project Glasswingが実際に発見した脆弱性の実績が公開されている。

OpenBSD 27年越しバグ Linux カーネル特権昇格 複数のゼロデイ発見

特にOpenBSDのバグは、セキュリティに定評のあるOSに長年潜んでいたという点で、セキュリティコミュニティに大きな衝撃を与えた。

Claude Mythos Preview ── 次世代モデルの片鱗

Project Glasswingと同時に発表されたのが、新モデル「Claude Mythos Preview」だ。コード理解とセキュリティ分析に特化した能力を持つとされ、Glasswingの中核エンジンとして使われている。

一般開発者向けの提供時期は未定だが、コード品質分析やセキュリティレビューの精度が大幅に向上する可能性がある。

💡 Claude Mythos Previewはまだ研究プレビュー段階。だが、将来的にClaude Codeに統合されれば、コードレビューの精度が次のレベルに到達する可能性がある。
